oFTP.Open()

Opens an FTP connection and returns the FTP object on success, 0 on failure

FTP_Open(Server, [Username, Password])

Parameters

Server

FTP server
Username

(Optional) Username
Password

(Optional) Password

Return Value

True on success, false otherwise.

Remarks

Port, Proxy/ProxyBypass, InternetConnectFlags can be set beforehand [see Properties in FTP_Init()]

Related

FTP_Init(), oFTP.CloseSocket(), oFTP.Close()

Example

oFTP.Open("ftp.autohotkey.net", "myUserName", "myPassword")
