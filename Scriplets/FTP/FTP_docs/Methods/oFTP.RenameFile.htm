oFTP.RenameFile()

Renames a file

oFTP.RenameFile(Existing, New)

Parameters

Existing

Existing file name, fully qualified path or relative path to current dir
New

New file name

Return Value

True on success, false otherwise

Remarks

none

Related

oFTP.DeleteFile()

Example

oFTP.RenameFile("MyScript.ahk", "MyTestScript.ahk")
