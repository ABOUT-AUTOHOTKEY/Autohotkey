oFTP.CloseSocket()

Closes session created by oFTP.Open

oFTP.CloseSocket()

Parameters

None.

Return Value

True on success, false otherwise

Remarks

The wininet module and wininet Internet open handles are not released.

Related

oFTP.Open(), oFTP.Close()

Example

oFTP.CloseSocket() 
;you can now create a new session with oFTP.Open
