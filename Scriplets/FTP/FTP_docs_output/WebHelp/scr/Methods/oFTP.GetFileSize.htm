oFTP.GetFileSize()

Renames a file

oFTP.GetFileSize(FileName [, Flags])

Parameters

FileName

Existing file name (fully qualified path or relative path to current dir)
Flags

See remarks

Return Value

Size of file in bytes (-1 on error)

Remarks

Flags:
FTP_TRANSFER_TYPE_UNKNOWN = 0 (Defaults to FTP_TRANSFER_TYPE_BINARY)
FTP_TRANSFER_TYPE_ASCII = 1
FTP_TRANSFER_TYPE_BINARY = 2

Related

oFTP.FindFirstFile(), oFTP.FindNextFile()

Example

oFTP.GetFileSize("MyTestScript.ahk", 0)
