Overview

Now read all your CHM e-books/files in your handheld PDA, like Palm, Pocket PC and smartphones in which iSilo Reader can be installed!!

This software will covert any CHM (Compiled Windows Help) files that you have, along with the navigation structure present in the document!

The user interface is very simple, infact you just have to point towards the source file and the destination folder!!

History

After searching through the net for a reader for CHM files, i was frustrated.
