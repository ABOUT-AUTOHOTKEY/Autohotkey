- Usage Instructions

 

Usage is pretty straightforward.

  • Run CHM2PDBConvert.exe.
  • Fill in the required feilds.
  • Hit "GO"
  • Hotsync the resultant PDB file to your handheld.

Tips & Tricks

If you want to change the settings of the Html to PDB conversion by iSiloX, make a backup of  "template.ixl" file in the root folder of this converter program and then edit template.ixl file in XML Notepad 2007! (Free download from Microsoft website, or just googel it)!
